II. Tutorial

 

Step 1. Selecting reconstructing data

 

In order to create models for a building façade, the TLS (Terrestrial laser scanning) data for this façade should be selected from a whole dataset first.

 

U can do this by either specifying the “map line”, or specifying a “reconstruction region”. Both approaches need an opened city map.

 

l        Create a new directory or enter an empty directory. Start the city modeler in this directory.

 

l        Open a city map. Click “File->Import->Map..”. Choose the map’s objects file and topologies file. (see Figure 1)

Figure 1. Importing map file

 

l        Specify the location of source data. (see Figure 2)

Figure 2 Set data source file

 

l        To select data by specifying map line, select a line from the opened map using the “select map line”  mode; or manually draw a line with “create new line”  mode. Click “Preprocess->Select reconstruction mapline” to start the selecting.

l        To select data by specifying region, simply use the “select area on canvas” mode  to draw a selection rectangle. Then click “Preprocess->Select reconstruction region” to start the selecting. (*Due to some unsolved problems, it is recommended not to use this method now.)

 

l        After selecting is completed, it is strongly recommended to save the laser point. (“File->Save->Laser points as…”)

 

Step 2. Reconstruct building model

 

Laser points need to be segmented before further processing. Note that the segmented laser point can be saved, so segmentation is only needed once.

 

l        Simply click “Segmentation->Surface growing” to segment laser points.

 

Then we need to find building features from laser points.

 

l        Click “Terrestrial->Find Feature” to start finding features.
